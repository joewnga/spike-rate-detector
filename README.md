# Spike Rate Detector 🦀

[![License: MIT OR Apache-2.0](https://img.shields.io/badge/license-MIT%20OR%20Apache--2.0-blue.svg)](LICENSE)
[![Rust](https://img.shields.io/badge/rust-1.80%2B-orange.svg)](https://www.rust-lang.org)

A fast, zero-dependency (pure Rust) spike-rate detector for any 1D time-series data.

Built for neuroscience, sensor data, financial tick analysis, heartbeat detection — anything with "spikes".

## Features

- Local-maximum peak detection with configurable threshold + refractory period
- Synthetic Poisson-spike data generator for easy testing
- Beautiful PNG visualization using `plotters`
